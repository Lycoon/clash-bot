Features

Player profile

It provides a lot of information about a player, such as his experience, townhall and troop levels. But also the clan he is in, trophies, league...

Warlog

The warlog shows various data about previous clan wars, indicating losses, wins, ties, destruction percentage, earned experience...

Clan profile

Similarly to the player profile, the clan profile shows all the necessary information known about a clan, such as its labels, badge, number of members...
