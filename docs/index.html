Clash of Clans
in your Discord server

Clashbot is a verified Discord bot aiming to provide Clash of Clans info from your favorite Discord server with simple commands. No more need to launch the game to see if your clan is winning the war it is in or succeeding in the current warleague round.

Highlights

User friendly

Intuitive commands and clear images respecting Clash of Clans interface

Free

Absolutely free of use. No paid membership or whatsoever

Multilingual

Clashbot comes in English, French, German, Dutch, Spanish and Portuguese

Verified

Officially verified Discord bot, which ensures quality and safety

Open source

Code has always been open source so anyone can take a look at it

Reliable

Technologies are performant and stable so the bot can be used at its full potential

Player profile

It provides a lot of information about a player, such as his experience, townhall and troop levels. But also the clan he is in, trophies, league, etc.

Warlog

The warlog shows various data about previous clan wars, indicating losses, wins, ties, destruction percentage, earned experience, etc.
